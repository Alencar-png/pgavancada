# Diretório de Dados

Este diretório contém o dataset utilizado no projeto.

## Dataset: Heart Disease

- **Arquivo:** `heart.csv`
- **Fonte:** UCI Machine Learning Repository / Kaggle
- **Link:** https://www.kaggle.com/datasets/fedesoriano/heart-failure-prediction

## Instruções para Download

1. Acesse o link acima ou o repositório UCI Machine Learning
2. Baixe o arquivo `heart.csv`
3. Coloque o arquivo neste diretório (`data/heart.csv`)

## Estrutura do Dataset

- **Instâncias:** 1025
- **Características:** 14
- **Tipo:** Classificação binária
- **Variável alvo:** HeartDisease (0 ou 1)

## Características

- Age: Idade
- Sex: Sexo (0 = F, 1 = M)
- ChestPainType: Tipo de dor no peito
- RestingBP: Pressão arterial em repouso
- Cholesterol: Colesterol sérico
- FastingBS: Glicemia de jejum
- RestingECG: Resultados do ECG em repouso
- MaxHR: Frequência cardíaca máxima
- ExerciseAngina: Angina induzida por exercício
- Oldpeak: Depressão ST
- ST_Slope: Inclinação do segmento ST
- HeartDisease: Presença de doença cardíaca (target)
